Bài 9 - Ứng dụng thực tế của lập trình mạng: Chat App, File Transfer, Network Monitor

Đăng ngày 22/10/2025 lúc 00:00

1. Giới thiệu

Lập trình mạng không chỉ là lý thuyết về mô hình OSI hay giao thức TCP/IP, mà còn là nền tảng để phát triển các ứng dụng thực tế trong đời sống và doanh nghiệp.
Ba ứng dụng tiêu biểu thể hiện sức mạnh của lập trình mạng gồm:

  • Chat App (Ứng dụng trò chuyện thời gian thực)
  • File Transfer (Truyền tệp giữa các máy)
  • Network Monitor (Giám sát và phân tích mạng)

Những ứng dụng này đều dựa trên nguyên lý client - server, socket và truyền dữ liệu qua giao thức mạng (TCP hoặc UDP).

2. Ứng dụng 1: Chat App (Ứng dụng trò chuyện)

2.1 Mô tả

Chat App là ví dụ điển hình của lập trình mạng, nơi nhiều người dùng có thể gửi và nhận tin nhắn theo thời gian thực thông qua server trung gian.

2.2 Cấu trúc

  • Server: Nhận và phân phối tin nhắn đến các client khác.
  • Client: Gửi tin nhắn và hiển thị nội dung từ server.

2.3 Mô hình hoạt động

  1. Client kết nối đến server qua TCP socket.
  2. Mỗi khi người dùng gửi tin nhắn, client sẽ gửi dữ liệu đến server.
  3. Server nhận tin nhắn và gửi lại cho tất cả client khác (broadcast).

2.4 Ví dụ Java – Chat Server

import java.io.*;
import java.net.*;
import java.util.*;

public class ChatServer {
    private static final int PORT = 12345;
    private static Set<Socket> clients = new HashSet<>();

    public static void main(String[] args) throws IOException {
        ServerSocket serverSocket = new ServerSocket(PORT);
        System.out.println("Chat Server đang chạy trên cổng " + PORT);

        while (true) {
            Socket clientSocket = serverSocket.accept();
            clients.add(clientSocket);
            new Thread(new ClientHandler(clientSocket)).start();
        }
    }

    static class ClientHandler implements Runnable {
        private Socket socket;
        private BufferedReader input;

        public ClientHandler(Socket socket) throws IOException {
            this.socket = socket;
            this.input = new BufferedReader(new InputStreamReader(socket.getInputStream()));
        }

        public void run() {
            try {
                String message;
                while ((message = input.readLine()) != null) {
                    System.out.println("Nhận: " + message);
                    for (Socket s : clients) {
                        if (s != socket) {
                            PrintWriter out = new PrintWriter(s.getOutputStream(), true);
                            out.println(message);
                        }
                    }
                }
            } catch (IOException e) {
                System.out.println("Client ngắt kết nối.");
            }
        }
    }
}

2.5 Chat Client đơn giản

import java.io.*;
import java.net.*;

public class ChatClient {
    public static void main(String[] args) throws IOException {
        Socket socket = new Socket("localhost", 12345);
        System.out.println("Đã kết nối tới server chat.");

        new Thread(() -> {
            try (BufferedReader in = new BufferedReader(new InputStreamReader(socket.getInputStream()))) {
                String response;
                while ((response = in.readLine()) != null) {
                    System.out.println(response);
                }
            } catch (IOException e) {
                System.out.println("Mất kết nối server.");
            }
        }).start();

        BufferedReader input = new BufferedReader(new InputStreamReader(System.in));
        PrintWriter out = new PrintWriter(socket.getOutputStream(), true);
        String msg;
        while ((msg = input.readLine()) != null) {
            out.println(msg);
        }
    }
}

3. Ứng dụng 2: File Transfer (Truyền file qua mạng)

3.1 Mô tả

Ứng dụng truyền file cho phép người dùng gửi tệp từ một máy đến máy khác thông qua kết nối TCP.
Server đóng vai trò nhận và lưu file, client là bên gửi.

3.2 Quy trình hoạt động

  1. Client chọn file cần gửi.
  2. File được đọc thành luồng byte và gửi qua socket.
  3. Server nhận luồng byte và ghi lại thành file mới.

3.3 Ví dụ Java – Truyền file

FileTransferServer.java

import java.io.*;
import java.net.*;

public class FileTransferServer {
    public static void main(String[] args) throws IOException {
        ServerSocket serverSocket = new ServerSocket(5000);
        System.out.println("Server sẵn sàng nhận file...");

        Socket socket = serverSocket.accept();
        InputStream in = socket.getInputStream();
        FileOutputStream fileOut = new FileOutputStream("received_file.txt");

        byte[] buffer = new byte[4096];
        int bytesRead;
        while ((bytesRead = in.read(buffer)) != -1) {
            fileOut.write(buffer, 0, bytesRead);
        }

        System.out.println("Đã nhận file thành công!");
        fileOut.close();
        socket.close();
        serverSocket.close();
    }
}

FileTransferClient.java

import java.io.*;
import java.net.*;

public class FileTransferClient {
    public static void main(String[] args) throws IOException {
        Socket socket = new Socket("localhost", 5000);
        File file = new File("send_file.txt");

        FileInputStream fileIn = new FileInputStream(file);
        OutputStream out = socket.getOutputStream();

        byte[] buffer = new byte[4096];
        int bytesRead;
        while ((bytesRead = fileIn.read(buffer)) != -1) {
            out.write(buffer, 0, bytesRead);
        }

        System.out.println("Đã gửi file: " + file.getName());
        fileIn.close();
        socket.close();
    }
}

4. Ứng dụng 3: Network Monitor (Giám sát mạng)

4.1 Mục tiêu

Ứng dụng giám sát mạng giúp phát hiện, thống kê và phân tích lưu lượng mạng.
Công cụ này thường được dùng trong:

  • Kiểm tra tốc độ mạng.
  • Phân tích gói tin (packet analysis).
  • Giám sát kết nối TCP/UDP đang hoạt động.

4.2 Các kỹ thuật sử dụng

  • Lớp InetAddress để xác định thông tin IP, hostname.
  • Lớp NetworkInterface để lấy thông tin card mạng.
  • Dùng ICMP Ping để kiểm tra kết nối.

4.3 Ví dụ: Kiểm tra tình trạng mạng

import java.net.*;

public class NetworkMonitor {
    public static void main(String[] args) throws Exception {
        InetAddress address = InetAddress.getByName("google.com");
        System.out.println("Đang kiểm tra kết nối tới: " + address.getHostName());

        if (address.isReachable(3000)) {
            System.out.println("Kết nối thành công!");
        } else {
            System.out.println("Không thể kết nối đến máy chủ.");
        }

        System.out.println("Địa chỉ IP: " + address.getHostAddress());
        System.out.println("Local Host: " + InetAddress.getLocalHost().getHostName());
    }
}

  • Kiểm tra trạng thái mạng và kết nối IP bằng Java.
  • Áp dụng mô hình TCP/IP vào bài toán cụ thể.

Hiểu và xây dựng được các ứng dụng này giúp bạn làm chủ nền tảng mạng, sẵn sàng phát triển những hệ thống phức tạp hơn như:

  • Dịch vụ đám mây và API phân tán
